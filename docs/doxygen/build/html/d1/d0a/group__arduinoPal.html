◆ TLE94112_PIN_CS1

#define TLE94112_PIN_CS1   10

Arduino Hardware Platform Pins.

This class provides a simple API for connecting the TLE94112 shield via SPI. The 12 halfbridges, the PWM units and the registers are set and controlled from the library.

Standard chip select pin for first TLE94112 shield
